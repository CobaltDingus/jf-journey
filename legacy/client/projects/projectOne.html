Our first project was to build a functioning game of Tic-Tac-Toe in the browser. All we had at this point was basic HTML, CSS, and JavaScript. For the game itself, the steps were relatively straightforward. Tic-Tac-Toe is played on a 3x3 grid so I created 9 divs and arranged them into a grid with CSS, using the aptly named ‘display: grid’ style. After some border styling to get the outlines, I had my board ready to go.
